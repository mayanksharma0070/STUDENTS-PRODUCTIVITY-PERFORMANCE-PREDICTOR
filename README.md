# Student Productivity & Performance Predictor (CLI)

## 📌 Project Description
This is a Command Line Interface (CLI) based project developed for the course *Fundamentals of AI and ML*.  
The system predicts a student's performance (marks) based on their study habits using a simple Machine Learning model.

The prediction is based on:
- Study hours
- Attendance
- Sleep hours

The project uses Linear Regression to learn from data and make predictions.

---

## 🎯 Objective
The main objective of this project is to understand how basic machine learning models work and how they can be applied to real-world problems like student performance prediction.

---

## ⚙️ Features
- Add new student data
- Train machine learning model
- Predict student marks
- View dataset
- CLI-based menu system
- Performance classification (Excellent / Average / Needs Improvement)

---

## 🧠 Technologies Used
- Python
- NumPy
- Pandas

---

## 📁 Project Structure
The project is organized into multiple files for better readability and modularity:

- <b>main.py</b> <br>
Handles the command-line interface, user input, and overall program flow.
- <b>model.py</b> <br>
Contains the machine learning logic, including training the model and making predictions.
- <b>utils.py</b> <br>
Includes helper functions such as menu display and performance classification.
- <b>data.csv</b> <br>
Stores the dataset used for training the model.
- <b>README.md</b> <br>
Provides instructions on how to run and use the project.
